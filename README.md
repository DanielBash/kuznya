# КУЗНЯ

> Движок для создания текстовых MU* на русском.

Данный движок создан с упором на быстроту и интуитивность разработки. Скриптовый язык: python. Редактируйте код и контролируйте сервер в одном процессе.

## Особенности
1) Редактор == Сервер - ведите разработку и модерацию из терминала, движок tui-first
2) Полностью настраиваемый веб-клиент
3) Автоматические порты на ssh, telnet, websocket
4) Легкие в освоении абстракции

## Запуск проекта(linux)

### Запуск через скрипт-установщик(linux)

1) **Скачайте и запустите установочный скрипт**
```bash
bash <(wget -qO- https://raw.githubusercontent.com/DanielBash/kuznya/main/install.sh)
```

### Установка вручную(linux)
1) **Скачайте официальный репозиторий и перейдите в него**
```bash
git clone https://github.com/DanielBash/kuznya.git
cd kuznya
```

2) **Установите зависимости и запустите среду разработки**
```bash
pip install -r requirements.txt
cd src
python main.py
```
